# Herdstück Feedback

Herdstück ist eine Kochbuch-App, die ich gerade baue. Wenn du Zugang hast 
und dir etwas auffällt, ist das hier der richtige Ort.

## Was gehört hierher?

**Bugs:** Etwas funktioniert nicht.  
**Ideen:** Dir fehlt ein Feature oder du hast einen Verbesserungsvorschlag.  
**Design:** Irgendetwas fühlt sich komisch an.  
**Performance**: Etwas lädt (zu) lange oder crasht.  
**Alles andere:** Auch das.

## Issue erstellen

1. Oben auf "Issues" klicken
2. "New Issue"
3. Kurz beschreiben was passiert ist
4. Bei Bugs: wie lässt es sich nachstellen?
5. Screenshot hilft immer

Keine Passwörter oder persönlichen Daten eintragen. Das Repo ist öffentlich.
